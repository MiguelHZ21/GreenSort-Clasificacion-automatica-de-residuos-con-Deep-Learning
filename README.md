♻️ GreenSort AI
Clasificación automática de residuos con Deep Learning
🚀 Descripción

GreenSort AI es un sistema de clasificación automática de residuos basado en visión por computador y aprendizaje profundo.
El proyecto permite identificar diferentes tipos de basura —como botellas plásticas, papel, envolturas y latas metálicas— a partir de imágenes procesadas en un entorno controlado.

El sistema utiliza técnicas de segmentación de imágenes, preprocesamiento avanzado y redes neuronales convolucionales (CNN) para lograr una clasificación eficiente y precisa.

🎯 Objetivo

Desarrollar un modelo capaz de:

Clasificar residuos en diferentes categorías
Automatizar procesos de separación de basura
Servir como base para sistemas de reciclaje inteligente
🧠 Tecnologías utilizadas
Python 🐍
OpenCV (procesamiento de imágenes)
NumPy
TensorFlow / Keras (o PyTorch, ajusta según uses)
Deep Learning (CNN)
